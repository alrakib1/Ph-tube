Discuss the scope of var, let, and const

var

variable declared with 'var' is a functional scope. It can be access within function. We can reassign it's value. We can update or change it based on necessary. It can be declared without initializing but the default value will be "undefined". 'var' declaration is not a block-scoped. So it can lead to unexpected behavior within blocks like if statements or loops.

let

let is a block-scoped variable. We declare it by 'let'. We can not reassign it's values. Though we can not reassign the value but we can update it's value. It optionally initialize itself to a value. If you want to access it before declaring then it will throw "Reference Error". It's only available within the block it was declared.

const

const is also a block-scoped variable like 'let'. We declare it by 'const'. We can not reassign it's values. If you want to access it before declaring then it will throw "Reference Error". It's only available within the block it was declared.

Discuss Null and Undefined

Null

We can initialize variables or object properties to 'null' if they are yet to be assigned a value. If we don't get valid result or result is absent from a function then it will give Null. Null can be used for error handling. "Null is an Object in JavaScript". A null value in JavaScript is used for referring absence of any object value and if any function or variable returns null, then we can infer that the object could not be created.

Undefined

There's eight ways you can get undefined-
(1) Variable that is not initialized will give undefined.
(2) Function with no return.
(3) Parameter that is not passed on a function will be undefine.
(4) If return has nothing on the right side will return undefined.
(5) Property that does not exists on an object will give undefined .
(6) Accessing array elements outside of index range.
(7) Deleting an element inside an array.
(8) Set a value directly to undefined.

What do you mean by REST API?

Api means Application Programming Interface. It uses HTTP requests to access and use data. That data can be used to GET, PUT, POST and DELETE data types, which refers to the reading, updating, creating and deleting of operations concerning resources. REST API is also refers to RESTful web services and It's used for web development purpose. It can be created with programming language such as JavaScript or Python.
