Discuss the scope of var, let, and const

var

variable declared with 'var' is a functional scope. It can be access within function. We can reassign it's value. It can be updated. It can be declared without initializing but the default value will be "undefined". 'var' declaration is not a block-scoped. So it can lead to unexpected behavior within blocks like if statements or loops. Here hoisting done, with initializing as 'default' value

let

let is a block-scoped variable. We declare it by 'let'. It can not reassign values.
